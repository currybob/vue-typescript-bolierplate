# vue-typesript-boilerplate

This boilerplate is based on vue-cli ver 3

### install

```
git clone "https://github.com/likelionWonHo/vue-typescript-bolierplate.git" 'your-project-name'
```

### Contains

- axios with vue-axios
- vue-router
- vuex
- vuex-persistedstate
- typescript
- prettier
- element-ui for vue
